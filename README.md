# Reminder app with ReactJS

This is a reminder app in ReactJS.

## Installation
```
git clone https://github.com/nisicadmir/reactjs-reminder.git
```
```
cd reactjs-reminder
```
Run installation
```
npm install
```

## Using app
Run server
```
npm run
```
Go to localhost:3000
```
http://localhost:3000/
```
Enjoy
